1. Download and install NSIS from nsis.sourceforge.net
2. Run NSIS compiler
3. Ensure to have opendylan.chm - generated by hhc opendylan.hhp in htmlhelp - in this directory!
4. Run build-installer.bat
5. The installer is output to opendylan-${APPVERSION}-win32.exe
6. (partial) List of known bugs/todos is at the top of opendylan.nsi
